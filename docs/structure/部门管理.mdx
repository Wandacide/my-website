# 部门管理

部门管理用于维护链工宝内的企业组织层级，确保员工、岗位、培训任务和学习档案可以按真实组织关系进行归属和统计。

## 适用场景

- 新建、编辑或停用企业部门。
- 为员工分配所在部门，便于培训任务下发。
- 按部门查看学习进度、考试结果和合规档案。

## 基本操作

1. 进入链工宝管理后台，打开“组织管理”。
2. 选择“部门管理”，查看当前组织树。
3. 点击新增或编辑，填写部门名称、上级部门等信息。
4. 保存后，组织结构会同步用于员工导入、任务分配和数据统计。

## 建议

保持部门命名与企业内部组织口径一致，避免同一部门出现多个相近名称，影响后续统计和合规归档。
