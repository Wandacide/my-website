# 创建账号

本文介绍管理员如何在全员培训管理端创建登录账号。账号创建完成后，用户可通过账号密码登录系统，并按照已分配角色访问对应功能。

## 功能简介

「创建账号」用于维护企业内部用户的登录身份。管理员可为员工创建账号，配置所属部门、联系方式、岗位、账号状态和角色权限，从而建立完整的账号体系。

创建账号通常用于以下场景：

- 新员工或新管理员需要进入系统；
- 需要为培训、考试、组织管理等业务角色开通后台权限；
- 需要将用户纳入部门组织架构和权限管理体系。

## 操作前提

创建账号前，请确认：

1. 当前登录账号拥有「账号管理 - 创建账号」的新增权限；
2. 如需选择归属部门，需提前在「部门管理」中维护组织架构；
3. 如需分配角色，需提前在「角色管理」中完成角色和权限配置；
4. 已明确新账号的使用人、所属部门、岗位和权限范围。

## 操作流程

### 1. 进入创建账号页面

在左侧导航栏中，选择「账号管理」-「创建账号」，进入用户列表页面。

[创建账号列表占位图]

> 图片占位：创建账号列表页面截图，建议展示用户列表、筛选区和「新增」按钮。

### 2. 新增用户

点击页面上的「新增」按钮，打开「新增用户」弹窗。

[新增用户弹窗占位图]

> 图片占位：新增用户弹窗截图，建议展示基础信息、部门、角色和状态字段。

### 3. 填写账号信息

按照页面要求填写账号信息：

| 字段 | 是否必填 | 填写说明 |
| --- | --- | --- |
| 用户昵称 | 是 | 用户在系统中的显示名称，例如「张三」。 |
| 归属部门 | 否 | 选择用户所属部门，影响后续数据权限范围。 |
| 手机号码 | 否 | 输入 11 位手机号，可用于登录验证、通知或找回密码。 |
| 邮箱 | 否 | 输入用户邮箱，可用于邮件通知或找回密码。 |
| 用户名称 | 是 | 登录账号名，创建后不可修改，需保证唯一。 |
| 用户密码 | 是 | 设置初始密码，建议包含大小写字母、数字和特殊字符。 |
| 用户性别 | 否 | 按实际情况选择。 |
| 状态 | 是 | 选择「正常」或「停用」。正常状态下账号可登录。 |
| 岗位 | 否 | 选择预设岗位，例如「部门经理」「普通员工」。 |
| 角色 | 是 | 选择已配置角色，决定用户可访问的功能权限。 |
| 备注 | 否 | 填写补充说明，便于后续管理。 |

### 4. 保存账号

确认信息无误后，点击「保存」。系统完成账号创建，并返回用户列表。

创建成功后，建议管理员同步告知用户：

- 登录地址；
- 登录账号；
- 初始密码；
- 首次登录后的密码安全要求。

## 配置建议

- 用户名称创建后不可修改，建议使用统一命名规范，例如手机号、工号或企业统一账号；
- 角色建议按岗位职责配置，不建议直接赋予过高权限；
- 对临时账号或外部协作账号，建议设置清晰备注，并定期检查状态；
- 如用户暂时不需要登录，优先选择「停用」而不是删除。

## 常见问题

### Q：用户名称创建后可以修改吗？

A：不可以。用户名称是登录账号名，创建后固定不可修改。如需变更，建议新建账号并停用旧账号。

### Q：新建账号后为什么登录看不到菜单？

A：请检查账号状态是否为「正常」、是否已分配角色、角色是否启用，以及角色是否配置了对应菜单权限。

### Q：一个账号可以分配多个角色吗？

A：可以。一个账号可拥有多个角色，系统权限按并集生效。建议避免多个角色之间权限边界不清。
